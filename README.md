# create-ubuntu-vm

 make a ubuntu vm  on hyper-v using cloud image with powershell

## Initial Setup

- Install qemu converter to convert the cloud image
  > choco install qemu-img -y

- Downlaod oscdimg from microsoft and add locatoin to script
